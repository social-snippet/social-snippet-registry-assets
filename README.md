# SocialSnippet::Registry::Assets

## Installation

```bash
$ bower install social-snippet-registry-assets
```

## Build

```bash
$ npm install
$ gulp
-> generate assets.js, assets.css
```

### Requirements

Node.js: 0.10.x

#### gulp (Build / Test)

    $ gem install gulp

#### juicer (Test)

    $ gem install juicer
